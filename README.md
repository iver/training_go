# Instalación

* Clonar repositorio:
  ```console
  $ git clone https://github.com/iver/training_go.git
  ```
* Instalar requerimientos
  ```console
  $ make install
  ```
*Nota:* La instalación solamente funciona en Mac por el momento. Se requiere tener instalado [Homebrew](https://docs.brew.sh/Installation)
  
# Ejecutar

```
$ make start
```
